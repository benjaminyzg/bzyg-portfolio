## Reporting a Vulnerability

If you believe you have found a security vulnerability in this project, please report it privately. **Do not create a public issue.**

To report a vulnerability:
1. Use the [GitHub Security Advisory](https://github.com/benjaminyzg/bzyg-portfolio/security/advisories/new) feature to submit a private report.
2. Alternatively, email me at [your-email-address].

### What to expect
* **Acknowledgment:** I will acknowledge receipt of your report within 48 hours.
* **Triage:** I will investigate the issue and determine its potential impact.
* **Resolution:** If a vulnerability is confirmed, I will work on a patch and coordinate a disclosure timeline with you.
